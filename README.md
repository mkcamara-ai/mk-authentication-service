Clean. Secure. Scalable.

---

## 🛡 Core Features

* ✅ User Registration & Secure Login
* 🔑 JWT Authentication (Access + Refresh Tokens)
* 📩 Email OTP Verification
* 🔐 Password Hashing (bcrypt)
* 🚫 Rate Limiting & Account Lockout (Brute-force protection)
* 🛡 CSRF Protection (`csurf`)
* 🧼 Input Sanitization & XSS Protection
* 🧱 Secure HTTP Headers (Helmet)
* 🌍 CORS Configured
* 👨‍💼 Optional Admin Endpoints (User + Activity Logs)
* 💎 Glassy / Liquid UI with Smooth Animations

## 🚀 Quick Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_NEW_USERNAME/secure-auth-system.git
cd secure-auth-system
```

---

### 2️⃣ Backend Setup

```bash
cd backend
cp .env.example .env
# Add MONGO_URI, JWT secrets, SMTP credentials
npm install
npm run dev
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend → `http://localhost:5173`
Backend → `http://localhost:5000`
